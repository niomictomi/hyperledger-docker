# HLF Blockchain Explorer

A web-based explorer for Hyperledger Fabric blockchain network.

## Features

- 📊 **Dashboard**: Overview of blockchain network
- 📦 **Block Explorer**: View all blocks with details
-  **Transaction Flow**: Visualize 5-phase transaction lifecycle
- 🌐 **Network Topology**: View network components
